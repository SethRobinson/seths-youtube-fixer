🔎 Comment search

Used only for “Find in comments”. Hidden by default; click Show to reveal it. Stored locally in this browser; never synced or sent anywhere except Google’s API.
How to get a free API key (~3 min):
  1. Open the Google Cloud Console and create a project.
  2. Enable the YouTube Data API v3 — the only API you need.
  3. Wait a few minutes for it to activate. Until it does, the next step’s API restrictions list shows “No items to display” and you can’t finish.
  4. Create an API key (optionally restrict it to YouTube Data API v3) and paste it above.
You don’t need to set up the OAuth consent screen — this key only reads public data. It uses your own Google quota (10,000 units/day). A key can’t be obtained automatically — it must be created in your account.
“Find in comments” loads up to this many comments (and replies) before pausing with a Load more button. Comments aren’t stored anywhere — this only bounds one search’s time and API quota. Default 50,000; max 200,000.
