Seth's YouTube Fixer

Comment search

Used only for “Find in comments”. Stored locally in this browser; never synced or sent anywhere except Google’s API.
How to get a free API key (~3 min):
  1. Open the Google Cloud Console and create a project (or pick an existing one).
  2. Enable the YouTube Data API v3.
  3. Create an API key and paste it here.
The key is tied to your own Google quota (10,000 units/day). A key can’t be obtained automatically — it must be created in your Google account.

Feed

Feedback cache

How long a captured “Hate content / channel” token stays usable. Lower it if old tokens get rejected. Cache lives in this browser only and survives tab switches and restarts.

Reset

Deletes this extension’s local data — captured feedback tokens, the action log, and all settings (including any “Don’t show this again” choices). Does not touch your YouTube account.
