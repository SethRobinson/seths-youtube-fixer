Seth's YouTube Fixer

Comment search

Used only for “Find in comments”. Stored locally in this browser; never synced or sent anywhere except Google’s API.
How to get a free API key (~3 min):
  1. Open the Google Cloud Console and create a project (or pick an existing one).
  2. Go to APIs & Services → Library → YouTube Data API v3 and click Enable.
  3. Go to Credentials → Create credentials → API key, then copy the key here.
  4. (Optional) Click Restrict key → restrict it to the YouTube Data API v3.
The key is yours and tied to your own Google quota (10,000 units/day — plenty for personal use). A key can’t be obtained automatically; it has to be created in your Google account.
